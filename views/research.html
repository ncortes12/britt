Food MacGyver

This app is for those busy weeknights when you need to throw together a meal fast. Search by ingredients to include or exclude and cook time. Built using HTML, CSS, JavaScript, Materialize, JQuery and Firebase. After only six weeks of coding classes this was done as a group project. My contribution to the project was AJAX/API calls, setting up the Firebase database and creating a user login in function.
